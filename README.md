# shoot_with_robot
🤖 Robot Shooting Playground
A fun and interactive web project where you can control a "Cute Full-Body Robot" and practice your shooting skills! This project is built using HTML, CSS, and JavaScript.

🚀 Features
Cute Full-Body Robot: An interactive, animated robot character displayed on the screen.

Dynamic Shooting: Click anywhere on the screen, and the robot will automatically aim and fire towards your cursor position.

Visual Effects: - "Bang! ❤️" text appears and floats upwards upon firing.

Smooth bullet animation with trajectory calculation.

Responsive Design: Works smoothly across different screen sizes.

🛠️ How to Run (Setup)
Download or copy the project files to your local machine.

Open the index.html file in any modern web browser (Chrome, Edge, Firefox, etc.).

Simply click anywhere on the screen to start shooting!

💻 Tech Stack
HTML5: Used for structuring the robot and the game interface.

CSS3: Used for styling, layout, and smooth animations (Keyframes, Flexbox).

JavaScript: Handles the core shooting logic, trigonometry (Math.atan2 for aiming), and DOM manipulation.
